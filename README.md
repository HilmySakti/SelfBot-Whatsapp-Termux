## Getting Started

This project require NodeJS v12

### Install
Clone this project

```bash
$ git clone https://github.com/HilmySakti/SelfBot-Whatsapp-Termux.git
$ cd SelfBot-Whatsapp-Termux
```

Install the dependencies:

```bash
$ pkg install nodejs
$ pkg install imagemagick
$ bash install.sh
```

### Usage
Run the Whatsapp bot

```bash
$ node index.js
```
